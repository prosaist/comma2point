# Утилита Comma2point
При сохранении данных вещественные числа записываются в текстовые файлы с использованием точки или запятой в качестве символа десятичного разделителя, отделяющего целую часть числа от дробной. Некоторые программы умеют читать такие файлы, содержащие точки, например, Mathcad. Другие программы предпочитают файлы, содержащие запятые, например, Microsotf Excel. Когда файлов много и они часто обновляются, удобно использовать предлагаемую утилиту для замены запятых на точки или наоборот. Она принимает на входе имена файлов, разделённые пробелами. После сборки в папке .\Win64\Release создаётся файл comma2point.exe. Следует запустить в командной строке shortcut.cmd, который скопирует файл comma2point.exe в папку C:\Users\Public и добавит ярлык для запуска утилиты в проводнике. После этого можно выбрать один или несколько файлов в проводнике, нажать правую кнопку мыши, в появившемся контекстном меню выбрать пункт "Показать дополнительные параметы" > "Отправить" > "comma2point", утилита запустится и обработает выбранные файлы.
